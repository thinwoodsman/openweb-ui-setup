# openweb-ui-setup
Notes, scripts, and config files from getting 
 [open-webui](https://github.com/open-webui/open-webui) 
 configured properly

* tika-notes.txt : Installing Tika 3.2.3 on Fedora 42
* tika-config.xml : Config file for the same
* tika-app : Shell script wrapper for tika-app that sets all the ENV variables
* sqlite-json-notes.txt : Viewing and modifying JSON text fields in SQLite
* comfyui-notes.txt : Setting up Open-WebUI to use Comfyui for image generation
* sdxl-basic-image-generation-api.json : API JSON export of ComfyUI workflow for image generation
